# Модуль User (Пользователь) 👤

Этот модуль является «точкой правды» для всех данных о пользователе в приложении.

## Ответственность
- Хранение и сериализация общих данных пользователя.
- Синхронизация данных профиля с Cloud Firestore.
- Предоставление унифицированной сущности `UserEntity` для UI.

## Модели и Данные

- **Domain**:
  - `user_entity.dart`: Основной объект с полями `uid`, `name`, `email`, `photoUrl`, `bio`, `username`, `dateOfBirth`.
- **Data**:
  - `user_model.dart`: Реализация маппинга JSON/Firestore.
  - `toDomain()`: Метод конвертации модели в чистую сущность.

## Использование
Любой модуль, которому нужны данные о текущем или другом пользователе, должен ссылаться на этот пакет, чтобы избежать дублирования моделей.
