# Модуль Authentication (Аутентификация) 🔑

Модуль отвечает за процесс входа, регистрации и восстановления доступа в приложении.

## Технологии
- **Firebase Auth**: Основной провайдер аутентификации.
- **Google Sign-In**: Интеграция с Google-аккаунтами.
- **Flutter Bloc**: Управление состоянием (AuthCubit).

## Структура (Clean Architecture)

- **Data**:
  - `datasource.dart`: Прямое взаимодействие с Firebase API.
  - `auth_repository_impl.dart`: Реализация логики репозитория.
- **Domain**:
  - `i_auth_repository.dart`: Контракт для работы с данными.
- **Presentation**:
  - `cubit/`: Логика управления состоянием.
  - `auth_screen.dart`: Главный экран входа/регистрации.
  - `email_verification_screen.dart`: Экран подтверждения почты.

## Особенности
- **Автоматическая синхронизация**: После входа через Google или Email, данные синхронизируются с Firestore в фоне.
- **Email Verification**: Статус подтверждения почты проверяется при каждом запуске приложения.
