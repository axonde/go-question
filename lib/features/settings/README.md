# Модуль Settings (Настройки) ⚙️

Модуль для управления внутренними параметрами приложения и персональными данными.

## Возможности
- **Профиль**: Просмотр и редактирование (bio, username, photo).
- **Безопасность**: Смена пароля, выход из системы.
- **Интерфейс**: Настройка темы, выбор языка (в планах).

## Структура
- **Presentation**: `settings_screen.dart`, форма профиля.
- **Domain/Data**: (В разработке) Взаимодействие с модулем `user` для обновления атрибутов.
