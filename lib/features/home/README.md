# Модуль Home (Главная) 🏠

Центральный хаб приложения, точка входа после авторизации.

## Функциональность
- **Главный экран**: Карусели топовых мероприятий, лента новостей.
- **Навигация**: Управление вкладками (BottomNavigationBar).

## Структура
- **Presentation**: `home_screen.dart`, виджеты для отображения краткой информации о событиях.
- **Data/Domain**: (В разработке) Взаимодействие с API для получения персонализированных рекомендаций.
