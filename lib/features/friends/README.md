# Модуль Friends (Друзья) 👥

Этот модуль отвечает за социальное взаимодействие: поиск друзей, отправку приглашений и список контактов.

## Текущий статус
- **Стадия**: Проектирование UI (Presentation).
- **План**: Реализация поиска пользователей через Firestore и системы подписок.

## Будущая архитектура

- **Data**: Список друзей пользователя в Firestore (под-коллекция или массив `uids`).
- **Domain**: `FriendEntity` (uid, name, photoUrl, status).
- **Presentation**: Экран поиска и список друзей.
