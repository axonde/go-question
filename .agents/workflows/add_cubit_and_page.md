# Интеграция нового Экрана и Cubit-а

Если требуется только UI и логика в существующей фиче, используй этот воркфлоу.

1. **Создание State**
   - Реши, какой подход к State нужен: простые классы (как `AuthLoading`, `AuthError`) или Sealed/Freezed класс.
   - Опиши все состояния, не забыв `Initial`, `Loading`, и `Error`.

2. **Создание Cubit**
   - Создай кубит, принимающий на вход существующий Репозиторий фичи.
   - Используй `emit` для обновления состояний.
   - Обеспечь обработку `result.fold` для вызовов внешних методов.

3. **Реализация Экрана (Page)**
   - Создай `XxxPage` как `StatelessWidget`.
   - Оберни его или его child в `BlocProvider` или `BlocConsumer`, чтобы слушать новые состояния.

4. **Обновление DI и Роутинга**
   - Обязательно зайди в `lib/injection_container/injection_container.dart` и добавь метод `sl.registerFactory(() => ...Cubit(sl()));`.
   - Если этот экран должен открываться откуда-то, обнови Router/Навигацию проекта.
